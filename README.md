## git基本操作
1.直接更改master分支的内容，并push自己的更改到master分支
- git add -A 准备上传你所有的更改
- git status 查看git给出的更改信息
- git commit -m "提交信息" 给出你的更改说明
- git push　　提交更改　（或者git push origin -u master　设置默认提交更改到master分支）

2.与团队项目同步
- git fetch upstream
- git merge upstream/master
